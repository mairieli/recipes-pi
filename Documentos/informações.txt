=========Como "pedir" uma tarefa=========
Escreva da seguinte forma:
*`Quero a issue #numeroDaIssue`* 
A frase não importa, esse foi só um exemplo. Utilize o *, porque vai deixar em negrito,
e o ` porque vai ficar vermelho. Não coloque espaço entre os marcadores e a frase. Isso
vai ajudar a não passar despercebido o pedido de vocês.

=========Trabalhando com o GIT=========
Clonar o Repositorio:
git clone https://github.com/mairieli/recipes-pi.git

~Sempre que for trabalhar~

Atualizar o branch master:
git pull origin master

Criar um novo branch para trabalhar: (E mudar para ele):
git checkout -b branchTrabalhando

Fazer as modificações necessárias.

Fazer o commit das modificações
git commit -am "Realizado alterações"

Voltar para o master:
git checkout master

Atualiza-lo novamente, caso alguém fez modificações:
git pull origin master

Trazer as modificações do do branch branchTrabalhando para o master:
git merge branchTrabalhando

Com o merge feito, enviar as alterações para o repositório:
git push origin master

~Repetir o precesso~
